Nhập/Xuất & Lưu trữ dữ liệu

Toàn bộ dữ liệu đang lưu trong bộ nhớ trình duyệt của bạn. Bạn nên xuất file Excel để sao lưu định kỳ tránh mất dữ liệu.

Chưa chọn file

Hướng dẫn sử dụng & Sao lưu Excel

1. Cách tính Giá mua trung bình:

Khi bạn thêm một giao dịch Mua cho xe *A* với giá 100k, sau đó mua tiếp xe *A* với giá 120k, hệ thống sẽ tự lưu và hiển thị Giá trung bình của xe *A* là 110k. Khi bạn bán xe *A*, giá vốn để tính lợi nhuận sẽ tự động lấy giá trung bình 110k này làm mốc.

2. Cách tạo file CSV mẫu để nhập hàng loạt:

Hãy bấm nút "Xuất dữ liệu Excel (CSV)" trước để có một file CSV chuẩn mẫu. Mở file đó ra bằng Excel, nhập các cột tương tự rồi lưu lại ở định dạng CSV. Sau đó click "Nhập dữ liệu từ CSV" để nạp hàng loạt vào phần mềm.

3. Tải lại trang (F5):

Dữ liệu được lưu trữ tự động trên máy tính của bạn thông qua trình duyệt. F5 hay tắt máy đi bật lại dữ liệu vẫn được bảo lưu nguyên vẹn.

Hướng dẫn

1. File mẫu kê khai là gì?

Đây là file Excel theo mẫu của Bộ Tài chính (TT 152/2025/TT-BTC) dùng để kê khai doanh thu hộ kinh doanh. Bạn tải lên để hệ thống có thể map dữ liệu vào đúng ô sau này.

2. Cách sử dụng:

Chọn file .xlsx phù hợp từ máy tính. Tên file và thời điểm cập nhật sẽ được lưu lại. Trong tương lai, hệ thống sẽ hỗ trợ xuất dữ liệu trực tiếp vào file mẫu này.
